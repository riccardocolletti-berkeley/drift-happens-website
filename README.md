# Drift Happens

Companion website for *Drift Happens: An Empirical Study of Neural Architecture
Robustness to Temporal Distribution Shift*.

A static site presenting the temporal drift matrices, per-model results, and
dataset overviews for Yearbook, Amazon Reviews, and arXiv. The paper is included
as [`drift-happens.pdf`](drift-happens.pdf).

Open `index.html` locally, or browse the live site at
<https://riccardocolletti-berkeley.github.io/drift-happens-website/>.

Authors: Robin Holzinger, Riccardo Colletti (UC Berkeley EECS).
